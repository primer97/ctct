City Controller


a town growing controller script.

Highlights :
* avoid endless towns growing
* unlock cargos to make town growing
* two modes: competitive / cooperative
* non-linear growing rate.
* FIRS compatible (any economy mode)
* ECS vectors compatible
* AXIS compatible


Author : hpfx
Forum thread : http://www.tt-forums.net/viewtopic.php?f=65&t=70224

Dependency : none.
OTTD version : 12+
JGRpp version : 0.56.0+
